## Linear Regression Preprocessing Recommendation Engine

This project demonstrates a structured, analysis-only preprocessing pipeline for linear regression modeling.  
The notebook inspects feature characteristics and produces explicit preprocessing recommendations without modifying the data.

### Key Artifacts
- Numeric preprocessing recommendation summary (CSV)
- Categorical preprocessing recommendation summary (CSV)
- Multicollinearity analysis and drop guidance (CSV)
